## 5. 주요 조치 및 해결 내역 (Action Items)

### 5.1 LinkedIn 로케이터 정밀 매칭
- **이슈**: 기존 .reusable-search__entity-result-list 선택자가 최신 레이아웃과 맞지 않음.
- **해결**: 실제 페이지 구조 분석을 통해 .entity-result__content-container를 식별하고, 명시적 대기 로직(wait_for_selector)을 추가하여 안정성을 확보함.

### 5.2 Twitter(X) 세션 갱신 및 검증
- **이슈**: 봇 탐지로 인한 로그인 차단.
- **해결**: 실제 Chrome과 Persistent Context를 사용하는 전용 갱신 스크립트 및 Smoke Test 구축.

### 5.3 서버 라우팅 및 BOM 오류 해결
- **이슈**: Python 소스 코드 BOM 인코딩 문제 및 404 에러.
- **해결**: UTF-8 (No BOM) 형식으로 재작성 및 루트 경로 정적 파일 프록시 설정.
