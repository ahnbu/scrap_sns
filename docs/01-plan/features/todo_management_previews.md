# TODO 관리 UX 옵션 아스키코드 미리보기

이 문서는 TODO 관리 기능의 3가지 상태(일반, 대기, 완료)를 처리하기 위한 UX 옵션을 시각화하여 제공합니다.

---

### 옵션 1: 순환 토글 방식 (Cycle Toggle)
우측 상단의 단일 아이콘이 클릭 시마다 상태를 순환시킵니다.

```text
+------------------------------------------+
| [@] User Name          [Star] [StateBtn] |  <-- StateBtn 클릭 시 순환
|                                          |      (○) 일반 -> (●) 대기 -> (✅) 완료 -> (○)
| Content of the post goes here...         |
|                                          |
| [#AI] [#News]                            |
|                                          |
| __________________________ [View Original] |
+------------------------------------------+

* 상태별 아이콘 변화:
  - 일반: [radio_button_unchecked] (○) - 회색
  - 대기: [pending] (●) - 주황색
  - 완료: [task_alt] (✅) - 초록색
```

---

### 옵션 2: 등록 및 하단 체크 방식 (Split Control)
상단에서 할 일로 등록하고, 하단에서 완료 여부를 관리합니다.

```text
+------------------------------------------+
| [@] User Name          [Star] [AddTodo]  |  <-- AddTodo: 할 일 목록에 추가/제외
|                                          |      (클릭 시 하단 Todo 제어바 노출)
| Content of the post goes here...         |
|                                          |
| [#AI] [#News]                            |
|                                          |
| [ Toggle: DONE ] __________ [View Original] |  <-- 하단에 완료 스위치 배치
+------------------------------------------+

* 동작 흐름:
  1. 상단 [+] 아이콘 클릭 -> '할 일'로 지정
  2. 카드 하단에 [ ] 완료 체크박스 또는 스위치 생성
  3. 완료 시 하단 스위치 토글
```

---

### 옵션 3: 상태 배지 방식 (Status Badge)
카드 상단에 명확한 텍스트 배지를 배치하여 상태를 강조합니다.

```text
+------------------------------------------+
| [@] User Name     [ TODO ] [Star] [More] |  <-- 좌측 상단 또는 우측에 배지 배치
|                   ^^^^^^^^               |      (클릭 시 상태 변경 팝업 또는 토글)
| Content of the post goes here...         |
|                                          |
| [#AI] [#News]                            |
|                                          |
| __________________________ [View Original] |
+------------------------------------------+

* 상태별 배지 디자인:
  - 일반: (없음)
  - 대기: [ TODO ] - 배경 주황색
  - 완료: [ DONE ] - 배경 초록색, 취소선 적용 가능
```

| :--- | :--- | :--- | :--- |

| **가독성** | 아이콘 색상으로 구분 | 위치로 기능 구분 | 텍스트로 즉시 인지 |
| **추천 대상** | 미니멀한 UI 선호 | 체계적 관리 선호 | 명확한 상태 확인 선호 |
