> **요약**: 링크드인 사용자 데이터(JSON)를 Markdown 형식으로 변환하여 사람이 읽기 쉬운 보고서 형태로 생성합니다.

---

## 1. 개요

### 1.1 목적

JSON 형식으로 저장된 링크드인 포스트 데이터를 Markdown 형식으로 변환하여, 기술적인 지식이 없는 사용자도 내용을 쉽게 확인하고 리뷰할 수 있도록 합니다.

### 1.2 배경

- 현재 스크래핑 결과는 JSON 파일로 저장되어 있어 텍스트 에디터 없이는 가독성이 떨어짐.
- 특정 사용자의 활동 내역을 요약된 문서 형태로 공유하거나 보관할 필요가 있음.

### 1.3 관련 문서 및 파일

- 원본 파일 1: `output_linkedin_user/zoon-chang/python/linkedin_py_full_20260207.json`
- 원본 파일 2: `output_linkedin_user/gb-jeong/python/linkedin_py_full_20260207_total.json`
- 결과 파일 (예시): `output_linkedin_user/{user_id}/report_{date}.md`

## 2. 범위

### 2.1 대상 범위

- [ ] JSON 데이터의 메타데이터(사용자 ID, 추출 일시, 총 건수) 요약
- [ ] 개별 포스트 데이터 변환:
  - 작성자명
  - 작성 시간
  - 본문 텍스트 (줄바꿈 유지)
  - 이미지/미디어 URL (Markdown 이미지 태그 활용)
  - 포스트 원본 링크
- [ ] 한글 깨짐 방지 (UTF-8 인코딩 준수)

### 2.2 제외 범위

- 인터랙티브 요소 (댓글 접기 등)
- 데이터 필터링 (모든 데이터를 변환 대상으로 함)
