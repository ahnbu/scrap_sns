- **특이사항**: `python/` 폴더 내에 `.md` 파일(예: `linkedin_py_full_20260207.md`)이 남아있어 폴더가 완전히 삭제되지 않은 경우가 발견되었습니다. 이는 의도적으로 `.json` 파일만 마이그레이션 대상으로 설정했기 때문입니다. 수집 데이터의 무결성에는 영향이 없습니다.
