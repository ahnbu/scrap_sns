## 3. 원인 분석 결과 (Root Cause)
- **인용 트윗의 구조**: Twitter에서 다른 트윗을 인용(Quote)할 경우, 한 `article` 내에 두 개의 `data-testid="tweetText"`가 존재하게 됩니다.
- **기존 로직의 한계**: 단순히 첫 번째 `tweetText`를 가져오거나, 작성자 확인 없이 모든 텍스트를 수집할 경우 인용된 원본 트윗의 텍스트가 작성자의 텍스트로 오인되어 수집되었습니다.
- **영향**: 특히 인용된 트윗이 긴 'Note Tweet'일 경우, 여러 인용 트윗들이 모두 동일한 원본 텍스트를 가지게 되어 데이터 중복이 발생했습니다.
