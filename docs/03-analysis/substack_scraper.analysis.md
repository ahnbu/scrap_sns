## 3. 발견된 차이점 (Gap) 및 개선 방안

- **날짜 추출 문제**: 
    - 현상: 일부 Substack 게시글 상세 페이지에서 `time` 태그의 `datetime` 속성이 없어 `created_at`이 빈 값으로 저장됨.
    - 개선안: 아카이브 목록(`list.html`) 수집 시점에 이미 존재하는 `datetime` 정보를 미리 확보하여 상세 데이터와 병합하는 로직 추가 필요.
- **문법 오류 수정**: 구현 과정에서 발생한 f-string 및 문자열 리터럴 오류는 즉시 수정 완료됨.
